## Development

### Test

```shell
yarn test
```

### Lint

```shell
# Fix issues
yarn format

# Check for issues
yarn lint
```

### Start

```shell
# Start a local HTTP server
yarn dev
```

## Getting Started

### Local development

1. Rename `.env.example` to `.env` and populate the values
2. `yarn install`
3. `yarn dev`
4. Browse `localhost:8080/api-docs` to view API Docs
